** User Services

*** Search all Users

**** Method

GET

**** Path

=/parle/users/all=

**** Parameters

/none/

**** Return

#+BEGIN_EXAMPLE
[
  {
    "userId" : "number",
    "userName" : "string",
    "userPassword" : "string",
    "userStatus" : "string"
  }
]
#+END_EXAMPLE

*** Search User By Id

**** Method

GET

**** Path

=/parle/users/user/{userId}/=

**** Parameters

/Path Parameter:/

=userId=

**** Return

#+BEGIN_EXAMPLE
{
  "userId" : "number",
  "userName" : "string",
  "userStatus" : "string"
}
#+END_EXAMPLE
